# 🌦️ Weather Analysis

## Overview
This project analyzes daily temperature data using Python, Pandas, and Matplotlib. It visualizes temperature trends and identifies maximum and minimum temperatures.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
Daily Minimum Temperatures Dataset from GitHub.

## Features
- Load and explore weather data
- Visualize temperature trends
- Find maximum and minimum temperatures
- Basic data analysis
